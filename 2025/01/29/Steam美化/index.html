一、官网

千禧年: 一个优异非常的steam美化插件，虽然目前美化方案较少，但是每一个的制作方案都很全面并且具有现代化、简易化的有点，没有繁杂的设计。

Millennium.

二、安装

1.下载

要在 Windows 上安装 Millennium，我们使用 PowerShell 安装程序脚本。要运行该脚本，请打开 PowerShell，粘贴以下命令，然后按 Enter。

Millennium 是完全可移植的，并且以下脚本不会改变任何系统配置。

2.自动安装插件

首先从名为this repository下载 Millennium 的 Windows 资产millennium-v0.0.0-windows-x86_64.zip。只需将所有文件放入您的 Steam 目录中，您可以通过运行下面的 PowerShell 命令找到该目录

3.手动安装插件

首先从名为 的this repository下载 Millennium 的 Linux 资源millennium-v0.0.0-linux-x86_64.tar.gz。提取并将所有文件放入 中~/.millennium/。然后使以下文件可执行

  • ~/.millennium/ext/data/cache/bin/python3.11（用于管理插件包；不是守护进程）
    ~/.millennium/ext/data/cache/bin/python3.11 （用于管理插件软件包；不是守护程序）
  • ~/.millennium/ext/bin/millennium
  • ~/.millennium/start.sh

通过运行chmod +x path/to/file

然后，您可以从~/.millennium/start.sh

或者，您还可以添加~/.millennium/ext/bin/以便$PATH能够全局运行 Millenniums 配置 CLI，尽管您也可以从 Steam 的 UI 配置 Millennium（Steam -> 设置 -> 插件和更新和主题）

三、选择方案

进入主题页面选择一个自己喜欢的方案，并按照他的安装教程进行安装

四、案例介绍

以SpaceTheme举例

1.自动安装（推荐）

  1. 下载Millennium。按照提供的说明安装修补程序。
  2. 下载SpaceTheme 安装程序或在 PowerShell 中运行命令：

2.手动安装

  1. 下载Millennium。按照提供的说明安装修补程序。
  2. 打开 Steam 并导航至Steam -> Settings -> Themes
  3. 单击文件夹图标打开皮肤文件夹
  4. 现在只需将主题放入皮肤文件夹中并解压 .zip
  5. 安装后，从界面中选择主题。
